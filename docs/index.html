Flux Browser

Flux is a lightweight Chromium-based desktop browser built with Electron. This is a developer preview.

Try Flux Locally

Flux is a desktop app. To try it:

git clone https://github.com/Flux3tor/flux-browser.git
cd flux-browser
npm install
npm start

View the source code on GitHub
